# Simon Memory Game - [Play the Memory Game](https://kilbeylyle.github.io/memory-game/)
![Resposive UI image](assests/imgs/readmeimgs/device_look.png)

## Overview

Simon Memory Game is an interactive browser-based game that challenges users to repeat an increasingly complex sequence of visual cues. The application focuses on event-driven logic and user interaction within a responsive interface.

## Tech Stack

- JavaScript
- HTML
- CSS

## Features

- Event-driven game logic
- Progressive difficulty through sequence generation
- User input handling and feedback
- Responsive browser-based interface

## Purpose

This project demonstrates core frontend development concepts including JavaScript logic, state management and user interaction handling.
